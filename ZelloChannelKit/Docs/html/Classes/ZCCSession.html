ZCCSession Class Reference

Inherits from NSObject
Declared in ZCCSession.h

Overview

ZCCSession represents the connection to the Zello Channels server. Each session you create connects to a single server and channel specified when the server is created. If you are connecting as a specific user, you must also specify the username and password when you create the session.

– startVoiceMessage

Creates and starts a voice stream to the server.

- (nullable ZCCOutgoingVoiceStream *)startVoiceMessage

Return Value

the stream that will be handling the voice message. If microphone permission has not been granted to the app, returns nil.

Discussion

Creates and starts a voice stream to the server.

The stream is created synchronously but started asynchronously, so it won’t actually begin transmitting until a session:outgoingVoiceDidChangeState: message is sent to the delegate.
