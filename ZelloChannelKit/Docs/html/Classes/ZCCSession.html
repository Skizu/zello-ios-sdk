Overview

ZCCSession represents the connection to the Zello Channels server. Each session you create connects to a single server and channel specified when the session is created. If you are connecting as a specific user, you must also specify the username and password when you create the session.

  delegate

Discussion

Delegate methods are called during connection and when messages are received from the channel. The session is ready for use after sessionDidConnect: is called.

– sendImage:

Sends an image message to the channel

- (BOOL)sendImage:(UIImage *)image

Parameters

image

the image to send

Return Value

YES if the image metadata was sent successfully. NO if an error was encountered before the image metadata could be sent.

Discussion

The Zello channels client will resize images that are larger than 1,280x1,280 to have a maximum height or width of 1,280 pixels. A 90x90 thumbnail will also be generated and sent before the full-sized image data is sent.

If an error is encountered while sending the image, the ZCCSessionDelegate method session:didEncounterError: will be called with an error describing what went wrong.
